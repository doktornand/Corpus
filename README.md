## 🌌 FONDATION ONTOLOGIQUE : LA TRAME FRACTURÉE

### Le Principe de Δ (Delta)

Dans l'univers de 2075, la réalité n'est pas fixe — elle est **tissée**.  
Le **Δ** représente l'écart entre :

- Ce qui *est* (couche **Lithique**)
- Ce qui *est dit* (couche **Mémétique**)
- Ce qui *est rêvé* (couche **Onirique**)
- Ce qui *doit être brisé* (couche **Glitchique**)

**Plus Δ augmente, plus le réel devient malléable.**  
Le Codex Vauvillensis est un **amplificateur de Δ** : chaque lecture modifie légèrement la trame.

### Les Trois Cœurs Moraux

| Cœur   | Nature                              | Effet sur la Trame          | Exemple                              |
|--------|-------------------------------------|-----------------------------|--------------------------------------|
| **Blanc** | Liberté, chaos créatif, mémoire vive | Augmente Δ, ouvre des possibles | Libérer un fragment corrompu        |
| **Noir**  | Contrôle, ordre absolu, oubli forcé | Réduit Δ, fige la réalité   | Effacer une mémoire "dangereuse"    |
| **Gris**  | Équilibre pragmatique, compromis    | Stabilise Δ, maintient le statu quo | Archiver sans interpréter       |

> *« Le danger n'est pas de choisir. Le danger est de croire qu'un seul cœur suffit. »*  
> — Azenor, 13ᵉ Siège du Vide, 2067

## 🏙️ CAEN-PROFONDE : ARCHITECTURE DES STRATES

### Le Principe de Superposition

Caen-Profonde n'est pas « sous » Caen. **Caen est au-dessus de Caen-Profonde.**  
La ville existe en **trois couches simultanées** :

```
CAEN-FANTÔME (1944)    ← Urd : le Passé figé
   │
CAEN-NEUTRE (2075)     ← Verðandi : le Présent négocié
   │
CAEN-PROFONDE (∞)      ← Skuld : le Futur potentiel

## 📚 LE CODEX VAUVILLENSIS : RÉSONATEUR, NON ARTEFACT

Le Codex n'est **pas un livre**, ni un fichier, ni un objet. C'est un **organisme informationnel temporel** :

- **Passif** : Il ne fait qu’**résonner**
- **Interactif** : Son contenu varie selon le lecteur, le moment et l’intention
- **Fragmenté** : Personne n’en possède la totalité — seulement des échos
- **Auto-référentiel** : Il contient des instructions pour se lire… et se modifier

### Structure Génétique du Codex

```
CODEX VAUVILLENSIS (génome temporel)
