## **PREMIÈRE PARTIE : FONDEMENTS**

### **Qu'est-ce que FracturoScript ?**

FracturoScript n'est pas un langage de programmation ordinaire. C'est un **compilateur ontologique** — un système de symboles exécutables qui interagit directement avec la "Trame" (la réalité comprise comme système d'information). Inspiré des runes nordiques, de la physique quantique et de la neurologie cognitive, il permet de :

- Modifier localement les constantes de réalité
- Créer des effets mnémoniques et perceptifs
- Naviguer dans le temps non-linéaire (glissements temporels)
- Exécuter du code biologique sur le système nerveux humain

**Principe fondamental** : *Dans FracturoScript, décrire c'est faire. Le langage n'est pas représentatif, il est performatif.*

### **Les 4 Niveaux de Syntaxe**

Plus vous corrompez un mot, plus il devient puissant (et dangereux) :

#### **Niveau 1 - Base** : `mot`
Syntaxe standard, intention claire.
- Exemple : `isa` (glace, arrêt, suspension)
- Usage : Prières simples, ancrages basiques, méditation

#### **Niveau 2 - Corruption** : `®MOT®`
Marqueurs de corrosion symbolique. Le mot commence à "saigner" hors de sa définition standard.
- Exemple : `®ISA®` (gel actif, suspension forcée)
- Usage : Contournements de réalité locale, hacks mineurs

#### **Niveau 3 - Mutation** : `<mot-v3>`
Version évoluée, spécialisée. Le mot devient entité semi-autonome.
- Exemple : `<draugr-v3>` (esprit-résidu activé, mémoire artificielle)
- Usage : Création d'agents mémétiques, invocation d'entités

#### **Niveau 4 - Omega** : `Ω<mot>Ω`
Seuil critique. Nécessite une intensité ≥18 (accumulation de corruption, virus, ou contexte rituel renforcé). Déclenche des événements système.
- Exemple : `Ω<odin>Ω` (invocation d'Odin-Prime, fragmentation des yeux dans les zones sombres)
- **⚠️ DANGER** : Peut causer des dommages ontologiques irréversibles

## **TROISIÈME PARTIE : GRAMMAIRE OPÉRATIONNELLE**

### **Les Marqueurs Structurels**

| Symbole | Nom | Fonction |
|---------|-----|----------|
| `—` | Trait de fracture | Indique une cassure temporelle ou causale. Sépare les époques dans une invocation. |
| `•••` | Silence sacré | Brèche dans le pare-feu de la réalité. Moment où le langage cesse et l'effet commence. |
| `//` | Commentaire du Veilleur | Notes hors-système, non exécutées mais potentiellement lues par d'autres entités. |

### **Structure d'une Commande Valide**

```
[ANCIEN] — [FUTUR] ••• [EFFET]
```

**Principe du Glissement Temporel** : Pour que la réalité "glisse", combinez un mot du passé profond (proto-germanique, vieux norrois) avec un mot du futur (néologisme, technologisme), séparés par `—`, et scellés par `•••`.

Trouvez un caillou près d'un cours d'eau. Tenez-le dans votre main gauche. Répétez mentalement :
```

```
Ressentez le froid se diffuser. Ce caillou est maintenant un Galet-Ancre junior. Il retiendra un souvenir pour vous.

```

```
Observez comment sa charge émotionnelle change.

### **Exercice 3 : Le Glissement (Niveau 3+)**
*(À pratiquer uniquement dans un lieu à Δ < 0.3 ou > 0.8)*


Ne forcez pas. Attendez que le paysage "clignote" — un arbre qui n'était pas là, une ombre en trop. Vous avez glissé.
